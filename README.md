# PharmaCare Inventory System

A web-based pharmacy inventory management system built using Java, JSP, Servlets, and PostgreSQL.

## Features
- User Login Authentication
- Add Medicine
- View Medicines
- Edit & Delete Medicines
- Dashboard with stock tracking

## Tech Stack
- Java (Servlets)
- JSP
- PostgreSQL
- JDBC
- HTML/CSS
